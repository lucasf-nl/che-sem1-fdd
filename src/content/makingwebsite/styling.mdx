# Styling

Om websites een kleurtje te geven en geavanceerdere layouts te implementered
wordt de programmeertaal CSS (Cascading Style Sheets) gebruikt. Omdat CSS
een nachtmerrie is wordt dit meestal vermijdt of vervangen. Deze website is
gemaakt met Tailwind CSS, een CSS library (deelbare en herbruikbare
verzameling code). Tailwind zorgt ervoor dat CSS als `flex flex-col gap-5`
wordt geschreven in plaats van

```css
.class {
  display: flex;
  flex-direction: column;
  flex-gap: 5em;
}
```
