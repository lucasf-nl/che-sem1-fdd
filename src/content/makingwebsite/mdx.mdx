# MDX

MDX is een markdown taal voor content. Het lijkt heel erg op markdown

```md
# Kop 1
## Kop 2
### Kop 3

**Dik gedrukt**, __onderstreept__, `code`

1. Genummerde lijst
2. Item 2

- Ongenummerde lijst
- Item 2
```

Dit heeft als resultaat

# Kop 1
## Kop 2
### Kop 3

**Dik gedrukt**, __onderstreept__, `code`

1. Genummerde lijst
2. Item 2

- Ongenummerde lijst
- Item 2

Maar ondersteunt ook het gebruik van geavanceerdere talen zoals YAML,
CSS, HTML, Javascript en Typescript, allemaal in 1 bestand. Dit maakt
het gemakkelijk om dit formaat te gebruiken om website inhoud op te slaan,
schrijven en verwerken.
