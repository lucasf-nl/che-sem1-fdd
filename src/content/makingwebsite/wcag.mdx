# Web Content Accessibility Guidelines

Mijn website gebruikt volledig semantische HTML voor zover mogelijk. Dit zorgt ervoor dat bijvoorbeeld
screenreaders de website duidelijk voor kunnen lezen voor de gebruiker. De elementen die niet semantisch
kunnen worden gedeclareerd krijgen ARIA rollen.
