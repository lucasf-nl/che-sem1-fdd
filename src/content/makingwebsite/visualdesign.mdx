# Visual Design

## Moodboard

Een moodboard geeft de vibe van een applicatie aan. Voor mijn moodboard heb ik simpel zwart, wit en grijs
kleurenschema met Helvectica Neue (Tailwind) voor lettertype. Ook heb ik een screenshot van een
[motherfucking website](https://motherfuckingwebsite.com) erin gezet, veel problemen met het moderne internet
zijn gewoon door website designers, ontwikkelaars en marketing gemaakt. Mijn website heb ik dus ook erg simpel
gemaakt.
