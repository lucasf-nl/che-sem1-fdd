# Astro

Astro is een JavaScript en JSX-framework ontworpen voor snelheid en een gemakkelijke
ontwikkelingservaring. Meestal draait Astro zonder server en compileert het naar
statische HTML, dit zorgt ervoor dat het vrijwel overal kan worden deployed,
inclusief Amazon S3. Astro biedt optimalisaties (zoals het inline zetten van SVG's)
en ondersteunt JSX components zodat een onderdeel, maar 1 keer hoeft worden geschreven.

Een handige functie waar ik veel gebruik van maak, is de ondersteuning van [MDX](./mdx).
Hierdoor kan ik content schrijven buiten de HTML, maar er ook HTML of Javascript in
zetten als ik dat wil.

Astro gebruikt de Island architectuur, origineel ontworpen voor E-commerce. Deze
techniek maakt het mogelijk om op een winkel bijvoorbeeld deze layout te hebben:

Met deze architectuur kunnen onderdelen als de header en footer (globale onderdelen in de
standaard layout), en de productinformatie (per product statisch in de HTML) meteen
beschikbaar zijn in de FCP (First Contentful Paint). De foto carousel en CTA's kunnen later
worden ge-rendered zodra bijvoorbeeld React beschikbaar is. Sommigen frameworks zoals React
ondersteunen hydration waardoor statische HTML later levendig kan worden gemaakt, maar het
voordeel van Astro hierbij is dat elk island in een ander framework kan worden geschreven.
