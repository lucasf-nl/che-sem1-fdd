# Pagina layout en content met HTML

HTML is een markup taal voor het maken van elementen op een website. Bij de ontwikkeling van deze
website is geen vanilla HTML gebruikt. Deze website bestaat uit een mix van Astro, React, Svelte en
MDX. Een groot nadeel van werken in puur HTML is dat er geen templating is, deze frameworks voegen
dat toe. Uiteindelijk wordt het geleverd naar de client als statische geminificeerde (kleiner) HTML,
met een beetje JavaScript voor React en Svelte.
